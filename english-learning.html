初中英语学习助手（人教版）

词汇 · 语法 · 练习 — 七年級~九年級全覆盖（适合基础薄弱同学）

学习路线图（七年級→九年級人教版全程）

核心原则：每天30分钟；按教材单元顺序推进；先词汇后语法；错题必回顾。本工具覆盖人教版七上~九年级全册词汇和语法。

初一篇：打好地基（七年级上/下共12单元）

学习重点

  • 词汇：七上 Starter~Unit 9：基础问候、家庭、学校用品、颜色、数字、月份、学科
  • 词汇：七下 Unit 1~12：能力爱好、日常作息、交通方式、校规、动物、天气、外貌、食物、周末活动
  • 语法：be动词 → 一般现在时(三单) → 人称/物主代词 → 现在进行时 → 一般过去时
  • 目标：掌握约600个基础词汇 + 4个核心时态，能读懂100词短文

初二篇：能力跃升（八年级上/下共20单元）

学习重点

  • 词汇：八上 Unit 1~10：假期旅行、频率副词、人物比较、影视娱乐、职业理想、未来预测、食物制作、邀请聚会、条件决策
  • 词汇：八下 Unit 1~10：健康急救、志愿服务、家务责任、人际沟通、过去事件、传说故事、地理自然、文学阅读、博物场馆、旧物情怀
  • 语法：比较级/最高级 → 一般将来时 → 情态动词 → 条件状语从句(if) → 现在完成时 → 过去进行时
  • 目标：掌握约800个进阶词汇 + 6个语法难点，能写80词短文

初三篇：冲刺中考（九年级全一册共14单元）

学习重点

  • 词汇：Unit 1~14：学习方法、节日文化、问路礼仪、成长变化、制造材料、发明创造、规则选择、推理判断、音乐偏好、风俗礼仪、情感感受、意外事件、环境保护、毕业回忆
  • 语法：宾语从句 → 被动语态 → 定语从句 → 情态动词表推测 → 综合复习
  • 目标：掌握约600个中考高频词 + 3大复杂句型，中考英语冲刺100+
